Usage: <cmd> sitename1,sitename2,sitename3 [limit]
Limits the number of simultaneous uploads per release on the specified site.
0 means unlimited.
Examples:
<cmd> * <- shows all current maxupperrip values
<cmd> HQ 3 <- use max. 3 simultaneous uploads per release on HQ
<cmd> HQ1,HQ2,HQ3 1 <- prevent dual uploading of the same release
See also: <cmdprefix>maxupdn

You can use the special sitename * as a wildcard for all sites.
