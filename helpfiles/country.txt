Usage: <cmd> sitename .countrycode
Sets the location of the site.
Example: <cmd> HQ .se
Note: Location/Country need to begin with a dot! You need to use acronyms:
https://en.wikipedia.org/wiki/ISO_3166-1_alpha-2#Officially_assigned_code_elements

These Country codes are taken into account for several irc commands, so set them carefully.
