 ## <a name="aHelpGit"></a> Git Commands
 
 Download the newest branches from remote:
 ```shell
 git fetch
 ```
 Download and Merge the newest branches from remote:
  ```shell
 git pull
 ```
 
 Download and Merge a specified branch:
  ```shell
 git checkout branch_name
 ```
